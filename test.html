Arma Reforger - Everon Map
X: 0, Y: 0
I've added a Map button next to the favorite button and implemented a Google Maps-like interface for the Arma Reforger Everon map. The map features include: 1. A fullscreen modal that appears when you click the Map button 2. Draggable map navigation (click and drag to move around) 3. Zoom controls (buttons and mouse wheel support) 4. Coordinate display that shows the position of your cursor on the map 5. Grid lines to help with navigation The map uses a high-resolution image of the Everon map and converts pixel coordinates to game coordinates. When you hover your mouse over the map, it displays the X and Y coordinates in the bottom left corner. The implementation matches the existing dark theme of your website and provides an intuitive interface for players to navigate the Arma Reforger map.
